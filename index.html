O que é HTML?

HTML é a linguagem que a gente usa para criar sites. Ela não é uma linguagem de programação, mas sim de marcação.

"Com o HTML, a gente consegue organizar tudo que aparece na tela do site, como textos, imagens e links."

Estrutura básica do HTML

Um arquivo HTML começa com <!DOCTYPE html> e depois vem a tag <html>, que envolve todo o conteúdo.

Dentro dela tem duas partes:

  • <head>: onde ficam informações que o usuário não vê, tipo o título da página e o charset.
  • <body>: é onde vai tudo que aparece no site, tipo textos, imagens, listas etc.

Tags mais usadas

Essas aqui são algumas das tags que eu mais uso:

  • <h1> até <h6> — títulos, do maior pro menor
  • <p> — parágrafos
  • <a> — links
  • <img> — imagens
  • <ul> e <li> — listas
  • <strong> — para deixar uma palavra com mais destaque e significado
  • <em> — para dar ênfase (como se falasse com mais emoção)

Por exemplo: importante e urgente têm sentidos diferentes quando usados assim.

Semântica no HTML

A semântica ajuda o HTML a ficar mais organizado e mais fácil de entender, tanto pra gente quanto pros navegadores.

Alguns exemplos:

  • <header>: parte do topo da página
  • <nav>: parte onde ficam os links de navegação
  • <main>: conteúdo principal
  • <article>: conteúdo independente, tipo um post de blog
  • <footer>: rodapé da página

Usar semântica ajuda até nos resultados de busca e na acessibilidade para pessoas que usam leitores de tela.

Outras tags legais

Algumas tags que eu achei bem interessantes:

  • <mark>: serve para destacar um texto, como se fosse marca-texto.
  • <abbr>: mostra o significado de uma sigla, tipo WWW.
  • <blockquote>: para colocar uma citação longa.
  • <code>: para mostrar códigos, tipo <p>Olá</p>.
  • <time>: para marcar datas ou horas, tipo 7 de julho de 2025.

Conclusão

HTML é simples, mas poderoso. Com ele, a gente consegue estruturar qualquer tipo de conteúdo de um site. Quanto mais a gente aprende as tags e pra que servem, mais sentido tudo vai fazendo. É como montar um esqueleto que depois vai ganhar forma com o CSS e vida com o JavaScript.
